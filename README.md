# üí´ About Me:
I'm currently leading and working on the frontend of different Gamindo's private projects: Dashboard, [Website](https://www.gamindo.com/), GAAS Configurator and a package of reusable components.

I developed even a file system analyzer for the frontend projects (configurable for many others, not only frontend) using Rust, that makes different structure in .json format to visualize immediatly with the help of some .json visualizer like [Json Crack](https://jsoncrack.com/), for internal use of Gamindo.
